Summary

This file develops the combinatorics of integer lattice boxes used in discrete geometry and combinatorics. It defines several representations of the d-dimensional box {0, …, n-1}^d as a finset of integer-valued functions, establishes bijections between these representations and finite types, and proves cardinality and inclusion results for both the standard box and the symmetric box {-n, …, n}^d. The material supports downstream applications such as pattern counting and sup-norm estimates for nested box regions.

Main definitions

Box Elements Bijection with Fin Functions (boxFnEquiv) (Figure 1) gives a computable bijection between elements of box d n and functions Fin d → Fin n.

Base-n Digit Decomposition to Lattice Point (boxIndex) (Figure 2) and Lattice point to base-n index (boxIndexInv) (Figure 3) are mutual inverses encoding lattice points in the box as flat natural-number indices via base-n digit arithmetic.

Symmetric integer box in d dimensions (symBox) (Figure 4) defines the symmetric discrete cube {-n, …, n}^d as a finset.

Box-to-Fin-power Equivalence (boxIxEquiv) (Figure 5) composes boxFnEquiv with finFunctionFinEquiv to obtain a bijection from box d n to Fin (n^d).

Main results

Cardinality of Combinatorial Box (box_card) (Figure 6) establishes |box d n| = n^d, and Cardinality of Symmetric Box (symBox_card) (Figure 10) gives |symBox d n| = (2n+1)^d.

Monotonicity of Box with Respect to Size (box_mono) (Figure 7) and Box Contained in Symmetric Box (box_subset_symBox) (Figure 11) are inclusion results, while Empty Box at Zero (box_zero) (Figure 8) handles the degenerate case.

Box Index Membership (boxIndex_mem) (Figure 9) confirms that boxIndex lands in the correct box.

The round-trip identities Box Index Inverse Round-Trip (boxIndex_boxIndexInv) (Figure 17) and Box Index Round-Trip Identity (boxIndexInv_boxIndex) (Figure 18) prove that boxIndex and boxIndexInv are mutual inverses; their agreement with the bundled equivalence is captured by Box Index Equivalence Value Agreement (boxIxEquiv_val) (Figure 13) and Box Index Equivalence Inverse Agrees with boxIndex (boxIxEquiv_symm_val) (Figure 14).

Sup-Norm Gap Between Nested Boxes (Lat.supNorm_sub_ge_of_inner_outer) (Figure 12) gives a quantitative distance lower bound between points in nested symmetric boxes, and Symmetric Box Disjointness with Set Difference (symBox_disjoint_sdiff) (Figure 15) records the resulting disjointness.

Finally, Cardinality of Pattern Type on Box Shape (fintype_card_pattern_eq) (Figure 16) counts assignments of an alphabet α over the box as |α|^(n^d).

def boxFnEquiv (d n : ℕ) : ↥(box d n) ≃ (Fin d → Fin n) where
  toFun v := fun j =>
    ⟨(v.val j).toNat, by
      have hv := v.property
      simp only [box, Fintype.mem_piFinset, Finset.mem_Ico] at hv
      obtain ⟨h1, h2⟩ := hv j
      have : (v.val j).toNat < n := by
        have h2' : v.val j < (n : ℤ) := h2
        have hnn : (0 : ℤ) ≤ v.val j := h1
        rw [Int.toNat_lt hnn]
        exact_mod_cast h2
      exact this⟩
  invFun f :=
    ⟨fun j => ((f j).val : ℤ), by
      simp only [box, Fintype.mem_piFinset, Finset.mem_Ico]
      intro j
      refine ⟨Int.natCast_nonneg _, ?_⟩
      exact_mod_cast (f j).is_lt⟩
  left_inv v := by
    ext j
    have hv := v.property
    simp only [box, Fintype.mem_piFinset, Finset.mem_Ico] at hv
    obtain ⟨h1, _⟩ := hv j
    show ((v.val j).toNat : ℤ) = v.val j
    exact Int.toNat_of_nonneg h1
  right_inv f := by
    ext j
    show (((f j).val : ℤ)).toNat = (f j).val
    simp

This definition constructs an explicit equivalence (computable bijection) between elements of the "box" box d n—functions Fin d → ℤ whose values lie in the integer interval [0, n)—and functions Fin d → Fin n. The forward map converts each integer value v(j) ∈ [0, n) to its natural number cast (which is bounded by n, hence a valid Fin n), while the inverse map casts each Fin n value back to an integer. The two inverse laws hold by the standard fact that toNat and natCast are mutually inverse on nonneg integers, combined with the boundedness conditions encoded in the membership of box d n.
