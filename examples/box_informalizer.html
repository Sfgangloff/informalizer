Summary

This file develops the combinatorial and geometric theory of lattice boxes — discrete hypercubic regions in $\mathbb{Z}^d$ or $\mathbb{N}^d$ — together with their cardinality properties and indexing schemes. It belongs to the intersection of combinatorics, discrete geometry, and formal fintype theory, providing foundational infrastructure for working with finite $d$-dimensional grids.

Main Constructions and Results

The file introduces two main types of boxes: the standard box box d n (the set of $d$-tuples of natural numbers with entries in ${0, \ldots, n-1}$) and the symmetric box symBox d n (the discrete hypercube ${-n, \ldots, n}^d \subseteq \mathbb{Z}^d$). Their cardinalities are established as $n^d$ and $(2n+1)^d$ respectively, and both constructions are shown to be monotone in their size parameter. A containment result (box_subset_symBox) relates the two: the box of side $n+1$ embeds into the symmetric box of half-width $n$. A key geometric lemma (Lat.supNorm_sub_ge_of_inner_outer) quantifies the $\ell^\infty$-distance between nested symmetric boxes: any point outside the shell $Q_{k+r}$ is at least $r+1$ away from any point inside $Q_k$, with an accompanying disjointness result (symBox_disjoint_sdiff).

Indexing and Equivalences

A central technical contribution is the base-$n$ indexing system: boxIndex d n i decodes a flat index $i \in {0, \ldots, n^d - 1}$ into a $d$-tuple via positional base-$n$ notation, while boxIndexInv is its inverse. These are shown to be mutual inverses, establishing a bijection between Fin(n^d) and box d n. This is packaged into explicit equivalences (boxIxEquiv, boxFnEquiv, patternFnEquiv) that identify the box with Fin d → Fin n or Fin(n^d), making it convenient to transfer computability and decidability. The file culminates in fintype_card_pattern_eq, which counts the number of $\alpha$-valued patterns on the box as $|\alpha|^{n^d}$, confirming that all the indexing machinery is consistent with the underlying fintype cardinality arithmetic.
