Summary

This Lean 4 file develops the foundations of symbolic dynamics in a general algebraic setting, formalizing the theory of subshifts over arbitrary groups (both additive and multiplicative). The main objects of study are configurations — functions from a group G to an alphabet A — and the shift actions, cylinder sets, patterns, and subshifts built from them.

Core Definitions and Structure

The file begins by defining the fundamental building blocks: the shift action (mulShift/addShift), which translates a configuration by left-multiplication by a group element, and cylinder sets, which are the basic clopen sets in the product topology on G → A, determined by fixing finitely many coordinates. The key lemmas (mulShift_one, mulShift_mul, continuous_mulShift, cylinder_eq_set_pi, mem_cylinder, isOpen_cylinder, isClosed_cylinder) establish that these constructions behave correctly topologically and algebraically. A subshift (Subshift/MulSubshift) is then defined as a closed, shift-invariant subset of the full configuration space, and the full shift mulFullShift (no forbidden patterns) is provided as the canonical example.

Patterns and the Forbidden Pattern Construction

The file introduces patterns (Pattern) as finite partial configurations — functions supported on a finite subset U ⊆ G, taking a default value outside U. Patterns generalize forbidden words from classical symbolic dynamics. The definition Pattern.mulOccursInAt captures when a pattern occurs in a configuration at a given position, and mulOccursInAt_eq_cylinder identifies such occurrence sets with cylinder sets, connecting the combinatorial and topological perspectives. The language of a subshift (LanguageOn, MulSubshift.languageOn) records which patterns appear in configurations of the shift. Given a family F of forbidden patterns, mulForbidden F is the set of all configurations avoiding every pattern in F at every position; this set is shown to be closed (isClosed_mulForbidden) and shift-invariant (mapsTo_mulShift_mulForbidden), so MulSubshift.ofForbidden packages it as a bona fide subshift. This realizes the classical fact that every sofic/SFT-type subshift can be defined by a set of forbidden patterns.

Overall Scope

The file covers the foundational layer of symbolic dynamics: topology of product spaces over discrete alphabets, the interplay between the algebraic shift action and the product topology, and the combinatorial notion of pattern occurrence used to define subshifts. The treatment is fully general (arbitrary group G, arbitrary alphabet A), handles both additive and multiplicative conventions via Lean's to_additive mechanism, and carefully tracks the left-cancellation hypotheses needed when G is not assumed to be cancellative. Together, these results provide a rigorous framework for studying shift spaces, their languages, and their forbidden-pattern characterizations in a theorem-prover environment.
